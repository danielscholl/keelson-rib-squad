This tier is the mental model behind Squad. It assumes you know the
[Keelson](https://danielscholl.github.io/keelson/) harness and spends its words on
what Squad adds.

:::note[Stub]
Scaffolded, not yet written. Planned pages for this tier:

- **Cast and roster** — project-driven team composition; casting specialists to fit
  the task.
- **The coordinator loop** — recall → assess → pick method → execute → reflect.
- **Method agency** — author a workflow, dispatch specialists, or modify code.
- **Governed memory** — shared decisions, per-agent memory, and the reflection pass.
:::

## Where a squad fits

| Mode | Use it for |
|---|---|
| **Chat** | Fast, open-ended work with one agent |
| **Workflow** | Repeatable execution with known steps |
| **Squad** | An issue-driven dev loop a coordinator runs and improves over passes |
